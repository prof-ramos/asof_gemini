# Documentation

This directory contains documentation for the ASOF project.

## Directory Structure

- **libraries/** - External library and framework documentation for reference
  - Next.js, React, TypeScript, Tailwind CSS, MDX, and other dependencies

## Documentation Overview

For project setup and development instructions, see the main [README.md](../README.md) in the root directory.

## Contributing

When adding documentation:
1. Place library/framework references in `libraries/`
2. Keep project-specific documentation in the root `docs/` directory
3. Use clear, descriptive filenames
4. Include code examples where appropriate
